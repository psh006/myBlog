# 搭建 VuePress文档

# 查看 node.js环境

前提条件 node.js版本需 >=8.6

C:\Users\Admin>node -v
v12.16.1

如果没有node.js环境,安装node.js,参考这里

# 开始搭建

全局安装 vuepress

C:\Users\Admin>npm install -g vuepress

cmd 进入想要搭建项目的目录下,创建项目目录,执行以下命令

PS D:\psh-workspace> mkdir bulid-vuepress

进入项目

PS D:\psh-workspace> cd bulid-vuepress

初始化项目，新建文件夹docs

PS D:\psh-workspace\bulid-vuepress> npm init -y

进入 package.json,修改脚本内容

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

进入docs,创建README.md文件,运行以下命令

D:\psh-workspace\bulid-vuepress>npm run dev

此时项目结构为

bulid-vuepress
├─docs
|  ├─README.md
├─package.json

建立基本目录

bulid-vuepress
├─docs
|  ├─.vuepress
|  |     ├─config
|  |         ├─nav.js
|  |         ├─sidebar.js
|  |     ├─config.js
|  ├─vue
|  ├─nuxt
|  ├─vuepress
|  ├─README.md
├─package.json

配置nav

module.exports = [
    // Nav 1
    {
        text: "主页",
        link: "/"
    },
    // Nav 2
    {
        text: "笔记",
        items: [
            {text: 'vue', link: '/vue/'},
            {text: 'nuxt', link: '/nuxt/'},
            {text: 'vuepress', link: '/vuepress/'},
        ]
    }
];

配置 sidebar.js

module.exports={
    "/vue/":require("../../vue/sideber_contents"),
    "/nuxt/":require("../../nuxt/sideber_contents"),
    "/vuepress/":require("../../vuepress/sideber_contents")
};
