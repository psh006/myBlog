# VUE服务端渲染(Nuxt.js)

# 搭建前准备

# 查看node.js环境

C:\Users\Admin>node -v
v12.16.1

如果没有node.js环境,安装node.js,参考这里

# 查看vue脚手架版本

C:\Users\Admin>vue -V
@vue/cli 4.2.3

如果vue脚手架低于3.0.0,应该删除当前版本,安装最新版本

npm uninstall vue-cli -g
npm install -g @vue/cli

如果没有安装过vue脚手架,直接安装最新版本

npm install -g @vue/cli

# 准备Nuxt脚手架

Nuxt.js 十分简单易用。一个简单的项目只需将 nuxt 添加为依赖组件即可。 为了快速入门，Nuxt.js 团队创建了脚手架工具 create-nuxt-app（确保安装了 npx（npx 在 NPM 版本 5.2.0 默认安装了））。 全局安装create-nuxt-app脚手架,在cmd窗口中输入以下命令

C:\Users\Admin>npm i -g create-nuxt-app

# cmd 进入想要搭建Nuxt.js项目的目录下,执行以下命令

PS D:\psh-workspace> npx create-nuxt-app nuxtdmo
